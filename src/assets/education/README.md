# Ảnh Education (slider)

Slider hiển thị 10 ô ảnh. Đặt tên theo số thứ tự `1`–`10`:

- `1.jpg`, `2.jpg`, … `10.jpg`

Định dạng: jpg / jpeg / png / webp / avif (Astro tự tối ưu WebP + lazy-load).
Ô nào chưa có ảnh → hiện `/placeholder.png`.

Muốn nhiều/ít hơn 10 ảnh: đổi số `length: 10` trong
`src/components/sections/Education.astro`.
